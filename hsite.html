HSite

HSite is my simple static website hosted with GitHub Pages. I'm not a web developer, so it's really just a minimal website to serve as the "frontend" for my projects. 

This site contains every project related information, announcements and binary downloads, just like GitHub, but in a friendlier way. Also it helps me to learn a bit about web development, because nowadays it's good to know it a bit at least.

If you're a developer, or you would like to learn, feel free to clone the source code of any of my projects and look into it, so maybe some of my solutions can help you.
