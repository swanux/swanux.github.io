HTidal 0.1.5 Beta released

Today I've finished working on the Snapshot 5 release of the program. It has nothing revolutionary new, but the user experience has improved a lot (at least in my opinion)

Changes:

  • Fixed minor bugs
  • Added new about section
  • Improved GUI usability
  • Updated deprecated code
  • Added play option to artist top/album/playlist views

You'll receive the update automatically if you've added hrepo to your system. Otherwise, you can download it manually.
