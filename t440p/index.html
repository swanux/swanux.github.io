T440p Hackintosh

This project helps to install macOS on a ThinkPad T440p.

It supports the following versions:

  • macOS 10.14.x - Mojave
  • macOS 10.15.x - Catalina
  • macOS 11.x - Big Sur

For more information and detailed guide, visit the project on GitHub.
