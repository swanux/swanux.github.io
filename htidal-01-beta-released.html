HTidal 0.1 Beta Released

I'm glad to finally announce my newest project: HTidal! If you would like to read more about it or view it on GitHub, you can visit the project page of it.

So let's talk about the release itself. It's the initial release of the program, so don't expect it to be completely stable and ready. It has nearly 100% functionality (home screen is missing) and I've tested it, so there shouldn't be serious bugs in it. If you encounter any problems, or you have any ideas then feel free to provide your feedback. 

You can either download the DEB package or install it from my APT repo to get automatic updates.
